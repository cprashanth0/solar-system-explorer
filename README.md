# Solar System Explorer

An interactive solar system built with React and Vite.
Click on planets to view information and explore orbital motion.

## Features

- Interactive planets
- Planet information popups
- Orbital animation
- Play / pause control

## Tech Stack

- React
- Vite
- JavaScript
- CSS
